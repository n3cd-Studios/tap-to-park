# tap-to-park

A parking application created with Go and Svelte.

## Getting Started

To install all of the packages, run `npm install`.
After that, you can:
- Run production: `npm start`
- Bring up the frontend: `npm run frontend`
- Bring up the backend: `npm run backend`
